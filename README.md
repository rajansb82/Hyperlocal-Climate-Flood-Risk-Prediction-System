**Explainable AI-Based Hyperlocal Climate Flood Risk Prediction System**

**Project Overview**

This project is a Machine Learning-based climate intelligence system that predicts flood risk at a hyperlocal level using environmental parameters.
The system combines Supervised Learning (Random Forest) and Unsupervised Learning (K-Means Clustering) to analyze climate patterns and classify flood risk with confidence scoring.
It also integrates Explainable AI (XAI) techniques to show feature importance, ensuring transparency in predictions.

**Features**

=> High /  Low Flood Risk Classification
=>Confidence Percentage Score
=> Heat Index Calculation with Severity Label
=> Feature Importance (Explainable AI)
=> Dynamic Risk Visualization (Color Alerts + Progress Bar)
=> Professional Dashboard UI
=> Hyperlocal Climate Input System

**Machine Learning Techniques Used**
 1) Supervised Learning
  =>Random Forest Classifier
  =>Probability-based prediction
2) Unsupervised Learning
  =>K-Means Clustering
  =>Climate pattern grouping

**Tech Stack**

Language: Python
Framework: Flask
ML Library: Scikit-Learn
Frontend: HTML, Bootstrap
Templating Engine: Jinja2

**Input Parameters**

The model takes the following environmental inputs:

=>Temperature (°C)
=>Humidity (%)
=>Rainfall (mm)
=>Wind Speed (km/h)

**System Workflow**

=>User enters climate parameters.
=>Random Forest model predicts flood risk.
=>Probability confidence is calculated.
=>Heat Index is computed for climate severity.
=>Feature importance is displayed for explainability.
=>Results are shown in a dynamic dashboard.
